📋 用户流程
握拳召唤 → 展开牌阵 → 食指选牌 → 双指翻牌 → 魔法特效（保留） → 新增：魔法揭示页 → 保存分享 / 继续翻牌
🌸 每张牌有专属内容
8张小樱牌各有不同的 魔法寓意文案（风=自由、火=勇气、水=包容、光=希望、影=智慧、花=美好、雷=决断、时=耐心），卡牌颜色对应光效变化。
📤 分享卡片生成
点「保存分享」→ 用 html2canvas 截图生成一张精美卡片 → 长按保存到相册 → 发朋友圈/小红书。卡片底部带 CYou Games 品牌 + 网址 + @羊仔coding不停，自带传播属性。
🔮 继续翻牌
点「继续翻牌」→ 回到牌阵继续玩。不强制结束，想翻几张翻几张，每张都能独立分享。
