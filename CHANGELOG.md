# Changelog

## 2026-04-09

- **UI**：缩小首页首屏留白与卡片尺寸（减少 `hero/showcase` padding、缩小网格间距与卡片内边距），并在桌面端将封面比例从 `3:4` 调整为 `4:5`，让三张游戏卡更容易一屏完整展示。
- **UI**：再次轻微缩小首页整体（标题字号/间距、卡片 padding、桌面端封面比例 `4:5 → 5:6`、按钮高度），进一步提升“一屏完整展示”稳定性。
- **Mobile UX**：游玩页改为“游戏优先”的底部抽屉简介（默认收起，点「简介」或拖拽把手展开），避免手机上简介挤占游戏画面高度。
- **WeChat**：检测微信内置浏览器（`MicroMessenger`）时在游玩页弹出提示，说明摄像头/手势可能无法正常工作，并引导用户「在浏览器打开」，提供一键复制链接。
- **Preview**：新增两个静态预览页用于确认「桌面主次层级」与「移动端上画面下介绍」布局（不影响线上默认样式）。
